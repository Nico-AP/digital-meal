Schritt 4/4: Daten hochladen

{# App instructions #}

Wenn Sie die Datei heruntergeladen haben, kehren Sie auf diese Seite hier zurück.

Klicken Sie weiter unten im Abschnitt «Datei auswählen» auf diesen Button:

Suchen Sie im Fenster, das sich öffnet, nach einer kürzlich heruntergeladenen ZIP-Datei, deren Name mit «TikTok_Data_» beginnt. Wählen Sie diese aus.

Beim Hochladen Ihrer Daten werden diese gefiltert. Jegliche Informationen, die Rückschlüsse auf Ihre Person zulassen, werden gelöscht. Dies geschieht auf Ihrem Gerät und es werden noch keine Daten übermittelt.

{# Browser instructions #}

Wenn Sie die Datei heruntergeladen haben, kehren Sie auf diese Seite hier zurück.

Klicken Sie weiter unten im Abschnitt «Datei auswählen» auf diesen Button:

Suchen Sie im Fenster, das sich öffnet, in Ihren Downloads nach einer ZIP-Datei, deren Name mit «TikTok_Data_» beginnt, und wählen Sie diese aus.

Ich kann die Datei nicht finden – was soll ich tun?

Wenn Sie die Datei im Safari Browser heruntergeladen haben, kann es sein, dass die ZIP-Datei automatisch entpackt wurde. Sie sehen dann entweder einen Ordner mit dem Namen «TikTok_Data_1234567890.zip» oder eine Datei mit dem Namen «user_data_tiktok.json».

Rechts-Klicken Sie auf den Ordner oder die Datei, und wählen Sie «…» komprimieren aus. Dadurch wird eine ZIP-Datei erstellt. Laden Sie anschliessend diese ZIP-Datei auf dieser Seite hoch.

Beim Hochladen Ihrer Daten werden diese gefiltert. Jegliche Informationen, die Rückschlüsse auf Ihre Person zulassen, werden gelöscht. Dies geschieht auf Ihrem Gerät und es werden noch keine Daten übermittelt.
