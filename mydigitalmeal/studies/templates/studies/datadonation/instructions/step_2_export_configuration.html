Schritt 2/4: Datenexport Konfigurieren

{# App instructions #}

Klicken Sie auf «Deine Daten herunterladen».

Wichtig: Wählen Sie als Dateiformat «JSON» aus.

Klicken Sie beim Punkt «Wähle Daten zum Herunterladen aus» auf «Alle auswählen».

Vergewissern Sie sich, dass Sie «JSON» als Dateiformat ausgewählt haben.

{# Browser instructions #}

Sie sollten nun eine Seite sehen, die «TikTok-Daten herunterladen» heisst und so aussieht:

Was tun, wenn ich auf einer anderen Seite gelandet bin?
Klicken Sie in der Menüleiste am Bildschirmrand auf «Profil».
Klicken Sie oben auf das Zahnrad.
Klicken Sie im Menü auf «Deine Daten herunterladen».
Nun sollten Sie eine Seite sehen, die «TikTok-Daten herunterladen» heisst und so aussieht:

Wichtig: Wählen Sie als Dateiformat «JSON» aus.

Klicken Sie beim Punkt «Wähle Daten zum Herunterladen aus» auf «Alle auswählen».

Vergewissern Sie sich, dass Sie «JSON» als Dateiformat ausgewählt haben.
